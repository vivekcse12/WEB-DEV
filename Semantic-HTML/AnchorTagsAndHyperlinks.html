Anchor Tags and Hyperlinks

What is an Anchor Tag?

The anchor tag (<a>) is used to create hyperlinks in HTML. It allows users to click and navigate to different pages or sections within a page.

Basic Structure of an Anchor Tag

<a href="URL">Link Text</a>

The href attribute specifies the destination URL of the link, and the text between the opening and closing tags is what users will see and click on.

Example of an Anchor Tag

Here is an example of an anchor tag that links to an external website:

<a href="https://www.example.com">Visit Example.com</a>

This will create a clickable link that says "Visit Example.com" and takes users to "https://www.example.com" when clicked.

Types of Links

  • External Links: Links that point to a different website (e.g., <a href="https://www.google.com">Google</a>).
  • Internal Links: Links that point to another page within the same website (e.g., <a href="/about.html">About Us</a>).
  • Anchor Links: Links that point to a specific section within the same page (e.g., <a href="#section1">Go to Section 1</a>).
